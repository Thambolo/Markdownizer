About

Markdownizer converts web pages to clean Markdown format using an AI-powered local agent. The extension captures page content (including authenticated views), and the agent intelligently compares multiple extraction methods to produce the best Markdown output.

Version: 1.0.0
Agent Required: Python 3.11+
Privacy: All processing happens locally—no data sent to external servers.
